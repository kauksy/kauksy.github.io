#### 1. 메소드는

- 메소드는 클래스가 가지고 있는 기능으로 클래스 안에 선언된다.
- 메소드는 입력 값이 있고, 그 입력 값을 받아 무언가 한 다음, 결과를 도출해 내는 수학의 함수와 비슷한 개념이다.
- 입력값: 매개변수, 결과값: 리턴값
- 인자(Argument): 어떤 함수 호출시, 전달되는 값
- 매개변수(Parameter): 전달 된 인자를 받아들이는 변수
